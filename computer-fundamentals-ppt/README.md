# Computer Fundamentals Presentation Generator

This Node.js application generates a professional PDF presentation covering computer fundamentals topics.

## Contents

The generated PDF includes:

### PRACTICAL - 1: Introduction to Computer and Its Organization
- What is a Computer?
- Computer Organization (Input Unit, CPU, Memory, Output Unit, Control Unit)
- Computer Characteristics (Speed, Accuracy, Diligence, Versatility, Storage, Automation)

### PRACTICAL - 2: Classification and Generations of Computers
- Classification of Computers (Supercomputers, Mainframes, Mini, Workstations, PCs, Embedded)
- Five Generations of Computers:
  - First Generation (1940-1956): Vacuum Tubes
  - Second Generation (1956-1963): Transistors
  - Third Generation (1964-1971): Integrated Circuits
  - Fourth Generation (1971-Present): Microprocessors
  - Fifth Generation (Present-Future): AI

### PRACTICAL - 3: Input and Output Devices
- Input Devices (Keyboard, Mouse, Scanner, Microphone, Webcam, Touchscreen, Joystick)
- Output Devices (Monitor, Printer, Speakers, Headphones, Projector, Plotter)
- Key Differences between Input and Output Devices

## Installation

```bash
npm install
```

## Usage

Generate the PDF presentation:

```bash
node generate-pdf.js
```

The PDF will be created as `Computer_Fundamentals_Presentation.pdf` in the current directory.

## Features

- Professional design with color-coded sections
- Clear headers and structured content
- Bullet points for easy reading
- Image placeholders for visual elements
- Comprehensive coverage of all three practicals
- Multi-page layout with proper formatting

## Dependencies

- `pdfkit`: PDF generation library
- `axios`: HTTP client for downloading images

## Output

The generated PDF includes:
- Title page with professional design
- Multiple pages covering all three practicals
- Well-formatted text with headers, sections, and bullet points
- Visual placeholders for diagrams and images
- Professional color scheme (Blue and Dark Gray)
